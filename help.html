Web Puzzle

Escolha o assunto desejado para saber mais!

Como funciona? O puzzle gera 100.000 mil endereços de bitcoins de formas completamente aleatórias para tentar encontrar uma combinação de alguma carteira puzzle.

Se essa verificação for bem sucedida irá te mostrar o endereço onde encontrará os bitcoins guardados.
Como converter para wif? Após ter conseguido o endereço gerado pelo puzzle, você precisará de um conversor WIF Offline.

Basta inserir o endereço gerado no conversor para obter o endereço completo.
Como resgatar bitcoins? 1. Obtenha uma carteira offline como a Electrum.

2. Acesse a carteira puzzle com o endereço já convertido em formato wif e transfira para sua carteira existente.
Dúvidas? Este código não irá roubar os teus bitcoins (apesar de ser MUITO DIFÍCIL DE ENCONTRAR BITCOINS) portanto fique tranquilo!

Se estiver com mais alguma dúvida me chame no GITHUB que está no coração anexado na página de configurações!
